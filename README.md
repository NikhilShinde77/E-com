# E-com
# 🛒 E-Commerce CRUD App (MERN Stack)

A full-stack **E-Commerce Product Management Application** built using **Node.js, Express, MongoDB, React (Vite)**.

This project allows users to **Add, View, Update, and Delete products** with a clean UI and proper routing.

---

## 🚀 Features

* 📦 Display Product List
* ➕ Add New Product
* ✏️ Update Existing Product
* ❌ Delete Product
* 🔗 Navbar with Routing
* ⚡ Fast UI using Vite
* 🌐 REST API integration

---

## 🛠️ Tech Stack

### Frontend:

* React.js (Vite)
* Axios
* React Router DOM

### Backend:

* Node.js
* Express.js
* MongoDB (Mongoose)

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/ecom-crud.git
cd ecom-crud
```

---

### 2️⃣ Backend Setup

```bash
cd backend
npm install
node server.js
```

➡️ Runs on: `http://localhost:5001`

---

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

➡️ Runs on: `http://localhost:5173`

## 🔌 API Endpoints

| Method | Endpoint                | Description      |
| ------ | ----------------------- | ---------------- |
| POST   | /api/product/create     | Add product      |
| GET    | /api/product/get        | Get all products |
| PUT    | /api/product/update/:id | Update product   |
| DELETE | /api/product/delete/:id | Delete product   |
